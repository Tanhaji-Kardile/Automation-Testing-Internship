# Automation Testing Internship Projects

## Overview

This repository contains Automation Testing projects completed as part of a QA Automation Testing Internship. The projects demonstrate hands-on experience in Web Automation Testing, API Automation Testing, CI/CD Integration, Security Testing Automation, Framework Development, and Automated Reporting using industry-standard tools and technologies.

---

## Projects Included

### 1. Data Driven Testing Using SauceDemo

This project automates the SauceDemo web application using Selenium WebDriver, Java, TestNG, and Data-Driven Testing techniques. Multiple login scenarios are executed using external test data to validate application behavior and functionality.

**Key Features:**

* Selenium WebDriver Automation
* Data-Driven Testing
* Login Validation
* TestNG Execution
* Extent Reports

---

### 2. E-Commerce Product API Testing (FakeStoreAPI)

This project focuses on API Automation Testing using FakeStoreAPI. Automated test scripts validate CRUD operations, response status codes, response payloads, and API functionality.

**Key Features:**

* REST API Testing
* CRUD Operations Validation
* Response Verification
* JSON Data Validation
* Automated API Execution

---

### 3. ParaBank CI/CD Test Automation

This project automates critical banking functionalities of the ParaBank application and integrates test execution with a CI/CD pipeline to support continuous testing and faster feedback cycles.

**Key Features:**

* Selenium Automation
* CI/CD Integration
* Jenkins Execution
* Automated Regression Testing
* Reporting and Build Validation

---

### 4. Web Application Security Testing Using DVWA & OWASP ZAP

This project automates security testing activities using DVWA (Damn Vulnerable Web Application) and OWASP ZAP. Automated scans are performed to identify security vulnerabilities and generate detailed security reports.

**Key Features:**

* Security Testing Automation
* OWASP ZAP Integration
* Vulnerability Detection
* Automated Security Scanning
* Security Report Generation
